# Playable Piano

## 🎹 Overview

Playable Piano is a web-based virtual piano application developed using HTML, CSS, and JavaScript. It enables users to play piano notes directly from their browser using either mouse clicks or keyboard shortcuts. The project demonstrates how to integrate audio playback, keyboard interactions, and dynamic user interface components to create an interactive musical experience.

Designed with responsiveness in mind, the piano works seamlessly across desktops, tablets, and mobile devices.

## ✨ Features

* Interactive piano keyboard
* Play notes using mouse clicks
* Keyboard key support for playing notes
* Real-time audio playback
* Adjustable volume control
* Show/Hide keyboard shortcut labels
* Responsive design for all screen sizes
* Smooth animations and visual feedback
* Lightweight and dependency-free

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Web Audio API

## 🚀 Getting Started

### Prerequisites

A modern web browser such as:

* Google Chrome
* Mozilla Firefox
* Microsoft Edge
* Safari

### Installation

Clone the repository:

```bash
git clone https://github.com/frankpatel1/Playable-PIANO.git
```

Navigate to the project folder:

```bash
cd Playable-PIANO
```

Open `index.html` in your browser.

## 🎵 Usage

1. Launch the application.
2. Click any piano key to play a note.
3. Use assigned keyboard keys for faster interaction.
4. Adjust the volume using the volume slider.
5. Toggle keyboard labels on or off.
6. Experiment with melodies and musical patterns.
